Titanic ML Pipeline

A supervised learning project for predicting Titanic passenger survival.

Python Pandas Scikit-learn Random Forest Logistic Regression GridSearchCV

Quick facts

Project type Machine learning assignment
Main stack Python, Pandas, Scikit-learn
Focus ML workflow and evaluation
Repository Public GitHub project

Problem / Goal

The goal was to build an end-to-end supervised learning workflow for predicting passenger survival using the Titanic dataset.

Key Features

  • Exploratory Data Analysis.
  • Missing-value handling.
  • Feature engineering, including FamilySize and FarePerPerson.
  • Random Forest and Logistic Regression models.
  • Hyperparameter tuning with GridSearchCV.
  • Evaluation using F1 Score and confusion matrix.

Technical Highlights

  • Built a full ML workflow from preprocessing to evaluation.
  • Compared multiple models.
  • Used feature engineering to improve the dataset representation.
  • Evaluated performance using classification metrics.

What I Learned

This project helped me practice the standard machine learning workflow, including cleaning data, building features, training models, tuning parameters, and interpreting evaluation results.
