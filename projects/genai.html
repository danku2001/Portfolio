HelmetGuard

A computer vision project that classifies construction-site images as safe or unsafe based on helmet usage.

Quick facts

Project type AI / ML academic project
Main stack Python, vision models, Stable Diffusion
Focus Synthetic data and safety classification
Repository Public GitHub project

Problem / Goal

Construction-site safety monitoring can be time consuming and inconsistent. The goal was to classify images as safe or unsafe based on whether workers are wearing helmets, while exploring synthetic data and real-world data.

Key Features

  • Created synthetic training data using Stable Diffusion.
  • Used real-world image data from public sources.
  • Built a safe / unsafe image classification dataset.
  • Fine-tuned pretrained vision models.
  • Compared ResNet18 and Vision Transformer approaches.
  • Used data augmentation to improve generalization.
  • Analyzed limitations, domain gap, and future improvements.

Technical Highlights

  • Combined generative AI and supervised image classification.
  • Compared model behavior on synthetic and real images.
  • Used pretrained models to improve training efficiency.
  • Presented results and limitations clearly instead of only reporting accuracy.

What I Learned

This project improved my understanding of computer vision, synthetic data, fine-tuning, model comparison, data augmentation, and real-world limitations in machine learning systems.
